# Contributing to Myah

## Development Workflow

1. Fork the repository
2. Create a feature branch (`feat/T3-{id}-description`)
3. Follow the development workflow in `AGENTS.md`
4. Run tests: `npm run test:frontend` and `pytest backend/`
5. Submit a pull request

## Code Style

- **Frontend:** TypeScript strict mode, tabs for indentation, single quotes, 100-char line width
- **Backend:** Python 3.11+, ruff formatting, 120-char line width, single quotes

## Commit Convention

Use conventional commits: `feat:`, `fix:`, `refactor:`, `chore:`, `docs:`

## Questions

Open an issue on GitHub for questions or feature requests.
